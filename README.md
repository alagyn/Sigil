# Sigil
A strongly-typed, object oriented, high-level language

The type safety of C++, combined with the syntactic sugar of Python.

## Progress:
 [x] Grammer Parsing  
 [x] LALR(1) Parse table generation  
 [ ] IELR(1) Parse table generation  
 [x] Token Scanner  
 [x] Base Parser  
 [ ] Syntax Tree generator  
 [ ] Assembly Generator  
 [ ] Build system
